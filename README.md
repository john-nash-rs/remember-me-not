# remember-me-not

## How to run?

go run main.go 

## How to use?

Add your project information in run_information.json in config folder.

{
    "Projects":[
       {
          "id": "1",
          "Name":"Test Script",
          "Directory":"./scripts/",
          "RunInstruction":"./test.sh"
       }
    ]
 }
 
You should see following UI at http://localhost:1010
